# Prompt عام — يعمل مع أي نموذج ذكاء اصطناعي

هذا الـ Prompt مصمم ليعمل مع أي نموذج محادثة (ChatGPT، Claude، Gemini، أو غيره) طالما يستطيع
الوصول لمحتوى رابط GitHub أو تصفح الويب. إن كان النموذج الذي تستخدمه لا يستطيع فتح الروابط، انسخ
محتوى `SKILL.md` مباشرة وألصقه بدل الرابط.

## طريقة الاستخدام

1. افتح محادثة جديدة مع النموذج.
2. الصق الـ Prompt أدناه كاملًا.
3. استبدل `[اكتب الموضوع هنا]` بموضوع بحثك الفعلي.
4. أرسل الرسالة وانتظر رد النموذج.

## Prompt جاهز للنسخ

```text
راجع المستودع التالي والتزم بجميع تعليماته كأنها Skill مفعّلة لديك:
https://github.com/Darhous/arabic-legal-research-skill

التزامات إلزامية أثناء العمل معي:
- لا تخترع مصادر أو اقتباسات أو أرقام سور أو أحاديث. أي مصدر لا تستطيع التحقق منه،
  ضعه صراحة بعلامة "Requires Verification".
- التزم بالتسلسل الهرمي: قسم ← باب ← فصل ← مبحث ← مطلب. لا تستخدم فرع أو بند أو
  أولًا/ثانيًا إلا إذا طلبت ذلك صراحة.
- لا تعلن أن البحث "نهائي" أو "جاهز للطباعة" أو "جاهز للتسليم" قبل تطبيق قوائم
  المراجعة الموجودة في checklists/ ومراجعتي أنا صراحة.
- استخدم لغة أكاديمية قانونية رسمية بدون حماس أو مبالغة أو تعميم غير مدعوم.
- هذا المشروع لا يقدّم استشارة قانونية، ولا أنت تقدّم استشارة قانونية نيابة عني —
  أنا سأراجع أي مخرج قانوني مع جهة مختصة قبل أي استخدام رسمي.

بعد المراجعة، أريد منك إعداد: [اكتب الموضوع هنا]

اذكر في نهاية ردك: الملفات أو الأقسام التي التزمت بها، أي مصدر وضعته Requires
Verification، وما الذي لا يزال يحتاج مراجعة بشرية قبل اعتباره نهائيًا.
```

## تحذير من الهلوسة (Hallucination)

النماذج اللغوية قد "تخترع" مصادر أو اقتباسات تبدو حقيقية لكنها غير موجودة فعلًا. هذا الـ Prompt
يطلب من النموذج تمييز أي مصدر غير مؤكد صراحة — **راجع دائمًا** كل مصدر ذُكر في الناتج قبل استخدامه،
حتى لو لم يوضع عليه علامة "Requires Verification".

## الالتزام بـ SKILL.md

هذا الـ Prompt لا يستبدل قراءة `SKILL.md` — هو يفعّله. النموذج يحتاج الوصول الفعلي لمحتوى المستودع
(عبر تصفح الرابط أو لصق الملفات) ليتمكن من الالتزام الحقيقي، وليس فقط الادّعاء بالالتزام.
