# Formatting Review Checklist

Use this checklist for DOCX and print-readiness review.

## Page Setup

- [ ] A4 page size.
- [ ] Print-ready margins.
- [ ] Page borders.
- [ ] Page numbers.
- [ ] Table of contents.
- [ ] Bibliography.

## RTL

- [ ] Arabic paragraphs are RTL.
- [ ] Headings are RTL.
- [ ] Footnotes are RTL.
- [ ] Footer is RTL from the right.
- [ ] Bibliography is RTL.
- [ ] URLs and Latin identifiers remain readable.

## Typography

- [ ] Arabic academic font selected.
- [ ] Headings are centered and bold.
- [ ] Paragraphs are justified.
- [ ] Paragraph indentation is consistent.
- [ ] Line spacing is controlled.

## Pagination

- [ ] Each `مبحث` starts on a new page.
- [ ] Each `مبحث` title appears centered at the top of the page in bold.
- [ ] No heading is orphaned at the bottom of a page.
- [ ] Footnotes appear at the bottom of the relevant page.

## Output Profiles

- [ ] Colored print version exists when requested.
- [ ] Black-and-white print version exists when requested.
- [ ] Both profiles preserve identical structure.
- [ ] Styling differences do not alter academic content.
