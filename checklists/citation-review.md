# Citation Review Checklist

Use this checklist before final delivery.

## Direct Quotations

- [ ] Direct quotation wording is preserved.
- [ ] Direct quotations are between quotation marks.
- [ ] Omitted words use `...`.
- [ ] Additions or corrections use `[ ]`.
- [ ] Each direct quotation is documented.
- [ ] Direct quotations are limited and necessary.

## Indirect Quotations

- [ ] Paraphrased material is rewritten in the researcher’s style.
- [ ] Paraphrased material is documented.
- [ ] Paraphrase does not copy the original structure too closely.

## Sacred and Famous Texts

- [ ] Quranic quotations preserve wording.
- [ ] Quranic quotations include surah and verse.
- [ ] Hadith quotations are verified from reliable sources.
- [ ] Famous quotations are verified and attributed.

## Internet Sources

- [ ] Author or responsible entity included where available.
- [ ] Page or article title included.
- [ ] Website or publisher included.
- [ ] URL included.
- [ ] Access date and time included.
- [ ] Missing metadata is marked.

## Integrity Failures

- [ ] No invented references.
- [ ] Unverified sources are marked `Requires Verification`.
- [ ] Footnotes relate to same-page content.
- [ ] Bibliography contains cited sources.
