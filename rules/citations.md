# Citation Rules

This file defines quotation and citation integrity requirements.

## Core Rule

Do not invent references. If real source verification is impossible, mark the source, citation, or footnote as `Requires Verification`.

## Direct Quotation

Direct quotation must:

- Preserve the original wording.
- Be placed between quotation marks.
- Be documented with a footnote or accepted citation mechanism.
- Be limited and necessary.
- Use three dots `...` for omitted words.
- Use square brackets `[ ]` for additions or corrections.

## Indirect Quotation

Indirect quotation must:

- Be rewritten in the researcher’s own style.
- Be documented.
- Preserve the meaning without copying the original structure mechanically.
- Avoid presenting another author’s idea as the researcher’s original conclusion.

## Quran, Hadith, and Famous Quotations

Quranic quotations must:

- Preserve wording exactly.
- Mention surah and verse.

Hadith quotations must:

- Be verified from reliable sources.
- Include documentation sufficient for academic review.

Famous quotations must:

- Be verified.
- Be attributed accurately.
- Not be used as decorative text when not connected to the research.

## Source Quality

Sources should be:

- Original whenever possible.
- Multiple.
- Diverse.
- Relevant.
- Suitable for legal academic research.

## Internet Sources

Internet references must include:

- Author or responsible entity, if available.
- Page or article title.
- Website or publisher name.
- Publication or update date, if available.
- URL.
- Access date and time.

If any required internet metadata is missing, record what is missing and mark the entry `Requires Verification` where appropriate.

## Citation Failure Conditions

Fail citation review if:

- A claim requiring support has no citation.
- A footnote cites a source unrelated to the page content.
- A reference appears fabricated.
- A direct quote lacks quotation marks.
- A paraphrase is too close to the source wording.
- Quran, Hadith, or famous quotation attribution is unverified.
