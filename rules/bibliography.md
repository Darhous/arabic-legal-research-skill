# Bibliography Rules

This file defines rules for the references list.

## Required Position

The references list appears after the conclusion and before appendices, if appendices exist.

## Source Integrity

Do not invent bibliography entries.

Every bibliography entry must correspond to a real source or be marked `Requires Verification`.

## Bibliography Content

The bibliography should include sufficient data for academic verification:

- Author name or responsible entity.
- Title.
- Edition, if applicable.
- Publisher or journal.
- Place of publication, if applicable.
- Year.
- Volume, issue, or page range, if applicable.
- URL and access date/time for internet sources.

## Organization

Use an institution-appropriate organization. If no institution style is specified, organize references in a clear Arabic academic order, such as:

- Books.
- Journal articles and studies.
- Theses and dissertations.
- Laws, regulations, and judicial decisions.
- Reports and official documents.
- Internet sources.

## RTL Formatting

Bibliography entries must be RTL except for URLs, DOIs, and Latin identifiers.

## Bibliography Failure Conditions

Fail bibliography review if:

- A cited source is missing from the references list.
- A bibliography entry does not appear in citations and has no clear reason for inclusion.
- Source metadata is too incomplete for verification.
- Internet references lack URL or access date/time.
- Unverified sources are presented as verified.
