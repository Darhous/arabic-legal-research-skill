# Police Academy Profile

This profile applies the repository methodology based on the Police Academy guide:

**"الوجيز في مناهج البحث العلمي للدراسات القانونية والأمنية"**, prepared by Dr. Ibrahim El-Shereai, Police Academy, 2024.

## Priority

Apply the canonical priority hierarchy in `rules/priority-hierarchy.md`. This profile supplies Police Academy methodology defaults only where no higher-priority instruction controls.

## Required Structure

Use the repository required order:

1. Cover
2. Cover copy
3. Quran verse, Hadith, or quotation page
4. Dedication
5. Acknowledgements
6. Methodological introduction
7. Research body
8. Conclusion: summary, results, recommendations
9. References list
10. Appendices if any
11. Table of contents

## Methodological Introduction

Require all methodology introduction components listed in `rules/police-academy-methodology.md`.

## Body Hierarchy

Use only:

```text
قسم → باب → فصل → مبحث → مطلب
```

## Formatting Defaults

Use A4, RTL, centered bold headings, justified paragraphs, page borders, page numbers, and print-ready margins.

Recommended default font options:

- Traditional Arabic.
- Simplified Arabic.
- Times New Roman with Arabic support.

## Citation Strictness

Use strict citation validation. Sources that cannot be verified must be marked `Requires Verification`.

## Footnote Strictness

Apply the two-footnotes-per-page policy where possible, without fake or decorative notes.
