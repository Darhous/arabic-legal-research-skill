# Examples

This directory currently contains no example fixtures. It documents the future example policy and fixture shape.

## Future Examples

- Minimal structured research input.
- Example approved plan fixture.
- Example unverified citation fixture.
- Example colored print output once DOCX generation exists.
- Example black-and-white print output once DOCX generation exists.

## Example Policy

Examples must be clearly illustrative. They must not present fabricated legal references as verified sources.

Use `Requires Verification` for illustrative citation records that are intentionally not verified.

## Minimal Example Shape

Future structured example fixtures should include:

- Title.
- Researcher metadata fields.
- Institution profile.
- Approved plan flag.
- Methodological introduction sections.
- Body headings using only `قسم → باب → فصل → مبحث → مطلب`.
- Footnotes connected to page content.
- Bibliography entries with verification status.

## Validation

Each example should include expected validation results. If a fixture intentionally fails, its failure purpose must be documented.
