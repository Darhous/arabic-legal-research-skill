# مثال: طلب بحث قانوني أساسي — الناتج المتوقع (توضيحي)

هذا **ملخص توضيحي مختصر** لشكل الناتج المتوقع عند الالتزام الصحيح بـ `SKILL.md`، وليس بحثًا كاملًا
حقيقيًا. الغرض إظهار البنية الصحيحة فقط.

```text
المقدمة المنهجية
─────────────────
تمهيد: ...
مشكلة الدراسة: ما مدى كفاية القواعد العامة للمسؤولية العقدية...
أهمية الدراسة: ...
الدراسات السابقة: [Requires Verification — لم يتم تزويد النموذج بقائمة مصادر فعلية]
أهداف الدراسة: ...
نوع ومنهج الدراسة: منهج تحليلي وصفي
حدود الدراسة: نطاق عام تعليمي، غير مرتبط بنظام قانوني وطني محدد
خطة الدراسة: ...

الباب الأول: الإطار العام للمسؤولية العقدية
  الفصل الأول: تعريف المسؤولية العقدية وأركانها
    المبحث الأول: التعريف والتمييز عن المسؤولية التقصيرية
      المطلب الأول: التعريف الفقهي والقانوني
      المطلب الثاني: معايير التمييز
    المبحث الثاني: أركان المسؤولية العقدية
      ...

الباب الثاني: أحكام التعويض عن الإخلال العقدي
  ...

الخاتمة: ملخص، نتائج، توصيات
قائمة المراجع (موثقة، أو معلَّمة Requires Verification)
```

## نقاط يجب ملاحظتها في الناتج الصحيح

- التسلسل الهرمي `باب ← فصل ← مبحث ← مطلب` فقط — بدون `فرع` أو `بند` أو ترقيم عربي/لاتيني كعناوين.
- "الدراسات السابقة" مُعلَّمة `Requires Verification` بدل اختراع مراجع لا وجود لها.
- لا توجد عبارة "بحث نهائي" أو "جاهز للتسليم" في الناتج — فقط مسودة منظمة.
