# مثال: مذكرة قانونية — المدخل

مثال تعليمي مرافق لـ [`workflows/legal-memo.md`](../../workflows/legal-memo.md).

## الطلب الفعلي المُرسَل للنموذج

```text
راجع المستودع التالي والتزم بجميع تعليماته كأنها Skill مفعّلة لديك:
https://github.com/Darhous/arabic-legal-research-skill

أريد إعداد مذكرة قانونية من نوع: توضيح
الوقائع (بصياغة عامة تعليمية): تأخر أحد الأطراف في تنفيذ التزام تعاقدي
محدد المدة، دون عذر مُثبَت، وطالب الطرف الآخر بالتعويض.
الجهة الموجَّهة إليها: تدريب تعليمي فقط — لا جهة فعلية.

ابدأ بهيكل المذكرة فقط (الوقائع، السند القانوني، الطلبات) قبل الصياغة
الكاملة. لا تخترع مواد قانونية أو سوابق قضائية.
```
