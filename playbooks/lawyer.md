# مسار المحامي

## لمن هذا المسار؟

لمحامٍ يريد مسودة أولية منظمة الشكل لمذكرة أو مراجعة قانونية أو تحليل موضوع، للتسريع فقط — **ليس
بديلًا عن رأيه القانوني المهني**.

## متى أستخدمه؟

- إعداد هيكل أولي لمذكرة قبل صياغتها النهائية بنفسك.
- مراجعة اتساق تنظيم مذكرة موجودة لديك (ترتيب، وضوح، توثيق مصادر).
- تلخيص أو هيكلة نصوص قانونية طويلة إلى تسلسل منظم.

## خطوات التنفيذ

1. **لا تُدخل بيانات عميل حقيقية حساسة** في محادثة عامة إلا وفق سياسة الخصوصية الخاصة بمكتبك.
2. استخدم الـ Prompt أدناه لطلب هيكل أولي أو مراجعة تنظيمية.
3. راجع كل نقطة قانونية بنفسك — المشروع يفحص **الشكل والاتساق فقط**، وليس صحة الرأي القانوني.
4. علّم كل نقطة تحتاج بحثًا قانونيًا إضافيًا منك قبل الاعتماد عليها.

## Prompt جاهز

```text
راجع المستودع التالي والتزم بجميع تعليماته كأنها Skill مفعّلة لديك:
https://github.com/Darhous/arabic-legal-research-skill

أريد مسودة أولية منظمة لمذكرة قانونية حول: [الموضوع/القضية بصياغة عامة
دون بيانات حساسة]

التزم بأسلوب قانوني رسمي، لا تخترع سوابق قضائية أو نصوصًا قانونية، وضع
"Requires Verification" على أي نص قانوني لم تتحقق من رقمه أو صياغته
الدقيقة. هذه مسودة عمل فقط، ولن أعتمدها كرأي قانوني نهائي دون مراجعتي
الشخصية الكاملة.
```

## أخطاء يجب تجنبها

- ❌ الوثوق برقم مادة قانونية أو نص سابقة قضائية دون التحقق من مصدر رسمي.
- ❌ إدخال بيانات عميل حقيقية أو حساسة في محادثة عامة غير مؤمَّنة.
- ❌ استخدام الناتج كرأي قانوني نهائي دون توقيعك ومراجعتك المهنية الكاملة.
- ❌ الاعتماد على "الشكل المنظم" كدليل على "الصحة القانونية" — هما أمران مختلفان تمامًا.

## مخرجات متوقعة

هيكل مذكرة أو تحليل منظم بأسلوب قانوني رسمي، بمواضع واضحة لكل نقطة تحتاج توثيقًا أو بحثًا إضافيًا
منك، جاهز لتستكمله وتوقّعه أنت شخصيًا.
