# مسار المستخدم العادي

## لمن هذا المسار؟

لأي شخص يريد بحثًا أو مذكرة قانونية عربية منظمة، بدون أي خلفية تقنية، وبدون تثبيت أي برنامج.

## متى أستخدمه؟

عندما تحتاج مسودة منظمة الشكل بسرعة — لمناقشة مع محامٍ، أو فهم أولي لموضوع قانوني، أو مسودة بحث
جامعي غير حاسمة بعد.

## خطوات التنفيذ

1. افتح ChatGPT أو Claude في المتصفح (لا حاجة لحساب مدفوع بالضرورة).
2. الصق الـ Prompt الجاهز أدناه.
3. استبدل `[الموضوع]` بموضوعك الفعلي بجملة واضحة ومحددة.
4. اقرأ رد النموذج كاملًا، وتحقق من قسم "ما يحتاج مراجعة" في نهاية رده.
5. إن أردت تعديلًا، اطلب تعديلًا محددًا (مثلًا: "أعد صياغة قسم أهمية الدراسة فقط") بدل "أعد الكل".

## Prompt جاهز

```text
راجع المستودع التالي والتزم بجميع تعليماته كأنها Skill مفعّلة لديك:
https://github.com/Darhous/arabic-legal-research-skill

أعدّ لي بحثًا قانونيًا عربيًا منظمًا حول: [الموضوع]

لا تخترع مصادر، وضع "Requires Verification" على أي مصدر غير مؤكد.
اذكر في النهاية ما يحتاج مراجعتي أو مراجعة محامٍ قبل أي استخدام رسمي.
```

## أخطاء يجب تجنبها

- ❌ اعتبار الناتج نهائيًا وتسليمه مباشرة دون قراءته كاملًا.
- ❌ تجاهل علامات "Requires Verification" وعدم التحقق منها بنفسك.
- ❌ استخدام الناتج كاستشارة قانونية رسمية — راجع [`docs/limitations.md`](../docs/limitations.md).
- ❌ طلب "بحث كامل نهائي جاهز للطباعة" من محادثة واحدة قصيرة — البحوث الجيدة تحتاج تكرارًا ومراجعة.

## مخرجات متوقعة

نص بحث أو مذكرة منظم بالتسلسل الهرمي الصحيح (قسم ← باب ← فصل ← مبحث ← مطلب)، بأسلوب أكاديمي رسمي،
مع قائمة مصادر وحواشي، وقائمة صريحة بما يحتاج مراجعة بشرية إضافية. هذا **ليس** ملف DOCX جاهزًا؛ إن
احتجت DOCX فعليًا راجع [`workflows/docx-production.md`](../workflows/docx-production.md) (يتطلب
تثبيت المطورين).
